### 水仙花数问题
水仙花数是指一个 n 位数 ( n≥3 )，它的每个位上的数字的 n 次幂之和等于它本身。本文将通过Python代码实现打印水仙花数，具体如下：
```python
#水仙花数
#narcissistic number
#水仙花数是指一个 n 位数 ( n≥3 )，它的每个位上的数字的 n 次幂之和等于它本身。
#（例如：1^3 + 5^3+ 3^3 = 153）
import math
import string
 
for x in range(1,10):
  a=x*x*x
  for y in range(0,10):
    b=y*y*y
    for z in range(0,10):
      c=z*z*z
      d=a+b+c
      w='%d' %x+'%d' %y+'%d' %z
      if d==int(w):
        print('水仙花数：'+w+'\n')
```
